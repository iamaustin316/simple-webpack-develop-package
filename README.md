# simple-webpack-develop-config
-------------------------------
simple webpack develop config file

### prepare list
`
    > npm install eslint -g
`

`
    > npm install webpack webpack-dev-server -g
`

`
    > npm install -g typescript typings webpack
`

`
    > npm install
`

`
    > npm link typescript
`

    > dist/
### entry root
    > index.html

### Star to begint

    > npm run dev
`
